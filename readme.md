# GitHubViewer  
**Первое Android-приложение за 1 день**  
*Просмотр профиля GitHub: аватар, био, репозитории, последний пуш*

![App Screenshot](screenshots/profile_torvalds.png)

---

## Описание

Простое и быстрое приложение для просмотра профиля пользователя GitHub.  
Вводишь имя — получаешь:

- Аватарку  
- Имя и био  
- Количество репозиториев и подписчиков  
- **Последний пуш** (дата, сообщение, репозиторий)

> Работает на **API GitHub v3** — никаких сторонних серверов.

---

## Используемые технологии

| Технология | Назначение |
|-----------|----------|
| **Android Studio** | IDE для разработки |
| **Java** | Язык программирования |
| **ConstraintLayout** | Адаптивный UI |
| **Retrofit 2** | HTTP-клиент для API |
| **Gson** | Парсинг JSON → POJO |
| **Glide** | Загрузка и кэширование аватарок |
| **GitHub REST API** | Данные пользователей и событий |
